# FocusFlow

O FocusFlow é uma aplicação desenvolvida para ajudar usuários a organizar suas tarefas e manter o foco no que realmente importa.

## Funcionalidades
- Criar tarefas
- Editar e excluir tarefas
- Marcar tarefas como concluídas
- Definir prioridades
- Interface responsiva

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Objetivo
Criar uma ferramenta simples e intuitiva para aumentar a produtividade no dia a dia.

## Futuras melhorias
- Login de usuários
- Sincronização com banco de dados
- Modo escuro
- Notificações
